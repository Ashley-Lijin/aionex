# Prompt for Building Symposium Theme Website

## Project Title
AIONEX 2K26 – Symposium Theme Page

## Goal
Create a modern, aesthetic, and mobile-responsive symposium landing page for **AIONEX 2K26**.  
The website should feel futuristic, tech-oriented, and energetic since it represents a college technical symposium.

Use **modern UI design**, smooth **hover animations**, and **GSAP animations** for dynamic transitions.

---

# Tech Stack
- HTML
- TailwindCSS
- JavaScript
- GSAP (for animations)

Design must be **fully responsive for mobile, tablet, and desktop**.

---

# Design Style
Theme: **Futuristic / AI / Tech / Fromal**

Use:
- Dark theme background
- Neon accent colors (blue / purple / cyan)
- Glassmorphism cards
- Smooth hover interactions
- GSAP entrance animations

Font suggestions:
- Headings: Orbitron / Space Grotesk
- Body: Inter / Poppins

---

# Page Structure

## 1. Hero Section (Landing Section)

The hero section should immediately show the event identity.

Content:
- Event Name: **AIONEX 2K26**
- Tagline: "Where Innovation Meets Intelligence"
- Event Date
- College Name
- Register Button
- Explore Events Button

Design requirements:
- Full screen hero
- Background gradient or animated particles
- GSAP animation on load
- Title text reveal animation
- Floating tech elements or glowing effects

Example layout:

Left side:
Event title and details

Right side:
3D illustration or abstract AI graphic

Animations:
- Fade in text
- Slide up buttons
- Parallax background

---

## 2. Events Section (Event Cards)

Display symposium events in **modern animated cards**.

Each card should contain:
- Event Name
- Short Description
- Student Coordinators
- Icon or illustration

Example Events:
- Hackathon
- AI Quiz
- Paper Presentation
- Coding Contest
- UI/UX Challenge
- Tech Debate

Card Design:
- Glassmorphism
- Rounded corners
- Soft shadow
- Neon border glow on hover

Hover Animation:
- Card lift
- Border glow
- Slight scale effect

GSAP Animation:
Cards should appear with **stagger animation when scrolling into view**.

Example card structure:

Event Card:
- Event Title
- 2–3 line description
- Coordinators list
- Register Button

Grid Layout:
Desktop: 3 columns  
Tablet: 2 columns  
Mobile: 1 column

---

## 3. Faculty Section (About the Staff)

This section highlights the faculty coordinating the symposium.

Each faculty card should include:
- Faculty photo
- Name
- Designation
- Role in symposium

Design:
Clean professional cards with hover effects.

Hover animation:
- Image zoom
- Card elevation
- Social icons appear

GSAP animation:
Fade-in + slide-up when scrolling.

Layout:
Desktop: 4 cards per row  
Tablet: 2 cards  
Mobile: 1 card

---

# Extra UI Features

Add the following micro interactions:

### Navigation
Sticky navbar with:
- Home
- Events
- Faculty
- Contact

Navbar should blur when scrolling.

### Buttons
Use glowing buttons with hover animation.

### Scroll Animations
Use GSAP ScrollTrigger:
- Section reveal animations
- Staggered card animations
- Smooth motion transitions

### Footer
Include:
- College Name
- Symposium Name
- Social media links
- Contact email

---

# Performance Requirements
- Fast loading
- Optimized images
- Smooth animations
- Clean code structure

---

# Final Goal
The final page should look like a **professional tech conference landing page**, similar to a startup event website.

It must feel:
- futuristic
- energetic
- interactive
- mobile friendly
- visually impressive
